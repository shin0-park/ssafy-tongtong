 도서관 나들이
├─ 01. 홈
│   └─ 서비스의 첫 진입 페이지
│
│   ├─ 1. 오늘의 추천 도서관
│   │   └─ 오늘의 기준 1개 + 기본 추천 도서관 3개
│   │
│   │      예시 기준
│   │      ├─ 오늘은 조금 넓은 도서관으로 가볼까요?(건물면적/부지면적)
│   │      ├─ 서가 사이를 천천히 둘러보기 좋은 날이에요(장서 수)
│   │      ├─ 분위기도 함께 즐겨보세요(공간/분위기 후기 태그)
│   │      ├─ 오늘은 차분히 앉아 있을 곳을 찾아볼까요?(열람좌석 수, 공부 관련 후기 태그)
│   │      ├─ 가족 나들이처럼 들르기 좋은 도서관이에요(어린이자료실, 가족/아이 후기 태그)
│   │      └─ 잠깐 쉬어가도 좋은 공간을 골라봤어요(라운지, 휴게공간, 쾌적함 후기 태그)
│   │
│   ├─ 2. 여기는 어때요?
│   │   └─ 로그인 + 선호/저장/후기 데이터가 있는 회원에게만 개인화 추천 3개
│   │
│   │      추천 근거
│   │      ├─ 사용자가 저장한 도서관
│   │      ├─ 사용자가 저장한 책
│   │      ├─ 사용자가 저장한 문화 프로그램
│   │      ├─ 사용자가 작성한 후기 태그
│   │      └─ 사용자 선호 목적 / 선호 지역 / 선호 시설
│   │
│   │      비로그인 또는 데이터 부족 상태
│   │      └─ 화면에 표시하지 않음
│   │
│   └─ 3. 테마별 추천
│       └─ 목적별로 도서관을 빠르게 탐색하는 추천 섹션
│
│          테마
│          ├─ 공부하기 좋은 곳
│          ├─ 책 보기 좋은 곳
│          ├─ 아이와 가기 좋은 곳
│          ├─ 분위기 좋은 곳
│          └─ 가까운 곳
│
│          동작
│          ├─ 테마 클릭
│          ├─ 해당 테마 조건으로 도서관 필터링
│          ├─ 필터링 된 데이터 상단의 6개 결과 표시
│          └─ 더보기 버튼 클릭시 도서관 찾기 페이지로 연결
│
├─ 02. 도서관 찾기
│   └─ DB에 저장된 도서관 정보를 제공하고 검색하고 필터링하는 페이지
│
│   ├─ 1. 검색
│   │   └─ 도서관명 / 지역 / 키워드로 검색
│   │
│   ├─ 2. 필터
│   │   ├─ 지역
│   │   │   └─ 부산광역시 구/군 기준
│   │   │
│   │   ├─ 도서관 유형
│   │   │   ├─ 공공도서관
│   │   │   ├─ 작은도서관
│   │   │   └─ 어린이도서관 등
│   │   │
│   │   ├─ 테마
│   │   │   ├─ 공부하기 좋은 곳
│   │   │   ├─ 책 보기 좋은 곳
│   │   │   ├─ 아이와 가기 좋은 곳
│   │   │   ├─ 분위기 좋은 곳
│   │   │   └─ 가까운 곳
│   │   │
│   │   ├─ 운영 조건
│   │   │   ├─ 오늘 운영
│   │   │   ├─ 주말 운영
│   │   │   ├─ 늦게까지 운영 (18:00 이후 운영)
│   │   │   └─ 공휴일 운영 / 휴관 여부
│   │   │
│   │   ├─ 시설/공간
│   │   │   ├─ 열람실
│   │   │   ├─ 어린이자료실
│   │   │   ├─ 디지털자료실
│   │   │   ├─ 주차장
│   │   │   ├─ 카페
│   │   │   ├─ 와이파이
│   │   │   ├─ 수유실
│   │   │   ├─ 장애인 편의시설
│   │   │   ├─ 엘리베이터
│   │   │   ├─ 휴게공간
│   │   │   └─ 야외공간
│   │   │
│   │   └─ 장서/열람실 조건
│   │       ├─ 자료 수
│   │       └─ 열람좌석 수
│   │
│   └─ 3. 결과 목록
│         ├─ 검색 혹은 필터 결과 미적용 상태에서는 전체 도서관 목록 표시
│         └─ 도서관 카드 리스트
│
├─ 03. 책 둘러보기
│   └─ 도서 검색 및 해당 도서를 보유한 도서관 목록을 제공하는 페이지
│
│   ├─ 1. 이번 주 인기 도서
│   │   └─ 정보나루 인기대출도서 API 기반 인기 도서 목록
│   │
│   ├─ 2. 도서 검색
│   │   ├─ 도서명 검색
│   │   ├─ 저자명 검색
│   │   └─ ISBN 검색
│   │
│   ├─ 3. 도서 상세 요약
│   │   ├─ 표지

│   │   ├─ 출판연도
│   │   ├─ KDC 기반 분류 태그
│   │   └─ 책 소개
│   │
│   └─ 4. 이 책을 보유한 도서관
│       └─ 선택한 도서를 소장한 부산 도서관 목록 제공
│

│   ├─ 1. 프로그램 검색
│   │   ├─ 프로그램명 검색
│   │   └─ 도서관명 검색

│   │   ├─ 지역

│   │   │   ├─ 전체
│   │   │   ├─ 강연/인문학
│   │   │   ├─ 독서/글쓰기

│   │   │   ├─ 체험/교육
│   │   │   ├─ 전시
│   │   │   └─ 기타

│   ├─ 3. 프로그램 카드 목록

│   │   ├─ 대상

│   │
│   └─ 4. 해당 도서관 상세 보기

│   │
│   ├─ 2. 후기 검색
│   │   ├─ 도서관명 검색

│   │   ├─ 본문 작성
│   │   │   └─ 짧은 후기, 최대 200자
│   │   └─ 태그 선택
│   │       └─ 필수 1~5개
│   │
│   ├─ 4. 후기 태그
│   │   ├─ 공부·열람

├─ 06. 나의 나들이

│   ├─ 1. 프로필 및 나의 나들이 요약

│   │   ├─ 닉네임을 이용한 사용자 인사 문구
│   │   ├─ 대표 선호 문장
│   │   └─ 선호 분석 기준 안내
│   │
│   │      예시
│   │      ├─ 김나들이님, 반가워요!
│   │      ├─ 조용한 학습공간과 장서가 풍부한 도서관을 자주 찾고 있어요.
│   │      └─ 저장한 도서관, 책, 프로그램, 후기 태그를 바탕으로 분석했어요.
│
│   ├─ 2. 나의 나들이 성향
│   │   └─ 최근 활동 데이터를 기반으로 방문 목적별 선호 비율 표시
│   │
│   │      성향 항목
│   │      ├─ 공부형
│   │      ├─ 책 탐색형
│   │      ├─ 프로그램형
│   │      └─ 휴식형
│
│   ├─ 3. 내가 많이 접한 태그
│   │   └─ 저장/후기/좋아요 데이터에서 자주 등장한 태그 표시
│   │
│   │      예시 태그
│   │      ├─ 조용한 공부 환경
│   │      ├─ 충분한 좌석
│   │      ├─ 다양한 장서

│   │      ├─ 카페

│
│   ├─ 4. 나의 관심 분야
│   │   └─ 저장한 책, 프로그램, 도서관 지역 데이터를 기준으로 관심 분포 표시
│   │
│   │      분석 항목

│   │      │   ├─ KDC 기반 분류

│   │          ├─ 저장한 도서관의 구/군 기준

│   ├─ 5. 저장한 도서관

│   ├─ 8. 좋아요한 후기

│   └─ 9. 내가 쓴 후기
